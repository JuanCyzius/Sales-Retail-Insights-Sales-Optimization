#Store Sales Analysis

An end-to-end sales analytics project using **SQL for data preparation and analysis** and **Power BI for building an interactive dashboard** to visualize key business insights from an electronics retail dataset.

---

# 📁 Project Overview

This project analyzes sales data from a multi-branch electronics retail store to uncover patterns in **sales performance, customer behavior, product demand, and payment trends**.

The goal was to transform raw transactional data into **actionable business insights** through SQL analysis and interactive Power BI visualizations.

---

# 🛠 Tools & Technologies

- **SQL (MySQL / SQL Server)** – Data cleaning and business analysis
- **Power BI** – Data visualization and dashboard creation
- **Excel / CSV Dataset** – Source data

---

# 📂 Dataset

The dataset contains **1,200+ sales transactions** from multiple store branches.

### Key Columns

| Column | Description |
|------|-------------|
| InvoiceID | Unique transaction ID |
| Store Branch | Branch location |
| City | Store city |
| Customer ID | Unique customer identifier |
| Customer Name | Customer name |
| Gender | Customer gender |
| Age | Customer age |
| Customer Type | Member / Non-member |
| Product Category | Category of the product |
| Product Name | Purchased product |
| UnitPrice | Price per unit |
| Quantity | Units purchased |
| Discount | Discount percentage |
| Tax | Tax percentage |
| Gross Amount | UnitPrice × Quantity |
| Discount Amount | Discount applied |
| Tax Amount | Tax added |
| Total Amount | Final transaction value |
| Payment Method | Cash / Credit Card / Ewallet |
| Purchase Date | Date of purchase |
| Satisfaction Rating | Customer rating (1–10) |

# 📊 Key Insights

Key Features & Insights:

-Profitability Analysis: Identification of high-margin product categories vs. high-volume drivers.
-Customer Segmentation: Comparative study between Members and Non-Members to evaluate loyalty program impact.
-Discount Optimization: Analyzing the correlation between discount rates, sales volume, and customer satisfaction ratings.
-Operational Dashboards: Interactive Power BI reports tracking KPIs across different store branches and payment methods.
